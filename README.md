# T9Search
Android T9 search.

提供T9匹配，不包含汉字拼音的生成。使用方法请看[示例][2]。

![preview][1]

===========================================================

Gradle

        compile 'cn.tinkling.t9:t9search:1.0'

Maven

        <dependency>
            <groupId>cn.tinkling.t9</groupId>
            <artifactId>t9search</artifactId>
            <version>1.0</version>
            <type>aar</type>
        </dependency>
